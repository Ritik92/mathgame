# Competitive Math Quiz

A real-time competitive math quiz where all users see the same problem and race to answer first. The first correct answer wins the round, and a new question is generated immediately.

## Features

* Live shared math questions for every connected user
* First correct answer is detected on the server
* New question appears automatically after each round
* Basic score tracking
* Handles different network speeds fairly

## Tech Stack

* Next.js
* Node.js
* WebSockets
* PostgreSQL
* Prisma ORM

## Getting Started

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Open the app at:
```
http://localhost:3000
```
